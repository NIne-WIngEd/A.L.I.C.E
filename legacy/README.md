# Legacy prototype

This directory is reserved for the original 2022 A.L.I.C.E. voice-assistant prototype.

The prototype is retained only as project history. It is not part of the current architecture, does not satisfy the permission model, and must not be run with real credentials or connected accounts.

The historic `mail.py` credential file is intentionally excluded and must never be restored. Future service integrations must use a dedicated secret manager, OAuth or scoped tokens, the permission gateway, and explicit approval flows.
